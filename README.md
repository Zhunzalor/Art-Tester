# Art-Tester

**Features**
1. Customize color
2. Load equipment weapon, armor, helm, cape, ground, and pet
3. Switch gender
4. Turn head
5. Show/Hide the background
6. Can save preview as an image
7. Dagger mode
8. Gauntlet mode
9. Can hide weapon, helm, cape, ground, and pet
10. Character size
11. Character emote
